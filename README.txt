TRUCK HUB — Landing Page V3 Premium / Clean Frontend Structure

Створено для передачі IT-спеціалісту.

Структура:
- index.html
- login.html
- css/main.css
- js/main.js
- assets/video/hero-video.mp4 — місце для майбутнього hero video
- assets/img/ — папка для майбутніх зображень

Що змінено:
1. HTML очищено від внутрішніх CSS і JavaScript.
2. Усі стилі винесено в css/main.css.
3. Уся логіка винесена в js/main.js.
4. Візуальна композиція збережена:
   - Hero Block 35% / 65%;
   - синьо-золота гама;
   - філософія часу одним стилем;
   - велика вантажівка як головний візуальний елемент;
   - Truck Hub Information Desk;
   - Truck Hub сьогодні;
   - блок під hero-video.mp4;
   - Як це працює.
5. Логотип тимчасово уніфіковано як текстова монограма TH, щоб на всіх сторінках був один і той самий знак.

Як запускати:
1. Розпакувати ZIP.
2. Відкрити index.html у браузері.
3. Для відео покласти файл hero-video.mp4 у папку assets/video/.

Примітка:
Це frontend-прототип. Backend, реєстрація, база даних, реальні аукціони та кабінети підключаються окремим етапом.
