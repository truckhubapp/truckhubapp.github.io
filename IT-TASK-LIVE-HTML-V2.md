# TRUCK HUB

## IT-TASK-LIVE-HTML-V2

### Завдання для IT-фахівця

Розмістити живий HTML-прототип Truck Hub V2 на тестовому сервері TrackHub.online або в окремій тестовій директорії.

## Файли

- `index.html`
- `customer-cabinet.html`
- `carrier-cabinet.html`
- `css/style.css`
- `js/app.js`
- `assets/truck-hero.svg`

## Перевірити

1. Відкриття `index.html`.
2. Блок `Truck Hub сьогодні`.
3. Кнопку `Показати більше`.
4. Мʼякий звуковий сигнал.
5. Кабінет Замовника.
6. Кабінет Перевізника.
7. Мобільну адаптацію.

## Принципи

- Truck Hub не банк.
- Truck Hub не бухгалтерія.
- Truck Hub не податкова.
- Truck Hub не HR-система.
- Truck Hub не Telegram.
- Truck Hub створює простір довіри та взаємодії.

Статус: **HTML Prototype / UX Prototype. Not production code.**
